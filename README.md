## Code in Master

Install the dependencies  
```bash
$ yarn install
```

Run the tests :heavy_check_mark:  
```bash
$ yarn test
```

### Building
BEFORE YOU COMMIT, MAKE SURE YOU REBUILD!!!

Build the typescript and package it for distribution
```bash
$ yarn build && yarn run pack
```
